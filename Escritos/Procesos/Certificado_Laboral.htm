Certificado Laboral

El Certificado de Empleados se utiliza para acreditar la relación laboral entre la empresa y el trabajador. Este documento permite certificar información básica del empleado, como su cargo, salario y tiempo de vinculación.

Antes de generar el certificado, es importante verificar que los datos del empleado estén completos, ya que el sistema utiliza esta información para construir el documento.
